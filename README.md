# 🇬🇧 Churchill — AI English Tutor Bot

Telegram-бот — AI-репетитор по английскому языку. Определяет уровень, строит персональную программу, объясняет темы, даёт упражнения, проверяет ошибки, тренирует слова и грамматику, ведёт от A0 до C1.

## Возможности (MVP)

- ✅ **Определение уровня** — детерминированный CEFR-тест (A0–C1) с 17 вопросами + AI для free-form
- ✅ **Персональный план** — автогенерация плана после теста, сохранение в БД
- ✅ **Уроки** — грамматика, лексика, упражнения с проверкой AI, персонализация по слабым местам
- ✅ **Учёт ошибок** — структурированная запись ошибок из уроков и диалогов, агрегация слабых мест
- ✅ **Прогресс** — статистика, серия дней, аналитика, дашборд со слабыми темами
- ✅ **Напоминания** — ежедневные push-уведомления, toggle on/off
- 🔤 **Словарь** — spaced repetition (SM-2), темы, повторение
- 💬 **Практика диалога** — свободное общение с AI на английском

## Стек

- **Python 3.12+**
- **aiogram 3** — Telegram Bot API
- **OpenAI API** (GPT-4o-mini) — AI-преподаватель
- **PostgreSQL** — хранение данных
- **Redis** — кеш
- **APScheduler** — напоминания
- **SQLAlchemy 2** + **Alembic** — ORM и миграции
- **Docker Compose** — деплой

## Быстрый старт

### 1. Клонировать

```bash
git clone git@github.com:krivoyandro/churchill.git
cd churchill
```

### 2. Настроить

```bash
cp .env.example .env
# Заполнить TELEGRAM_BOT_TOKEN и OPENAI_API_KEY
```

### 3. Запустить (Docker)

```bash
docker compose up -d
```

### 4. Запустить (локально)

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Нужен PostgreSQL
python main.py
```

## Текущий статус

**Реализованный E2E-сценарий:**

`/start` → выбор цели → выбор темпа → тест уровня (детерминированный + AI) → автогенерация плана → урок с AI → проверка ответа → запись ошибок → завершение урока → стрик + статистика → прогресс-дашборд.

**Готово (MVP 1):**
- Онбординг FSM: `/start` → цель (8 вариантов) → минуты (10/20/30/45/60) → тест
- Тест уровня: 17 детерминированных вопросов, early stop при 2+ ошибках на уровне, AI для free-form
- Автогенерация плана после теста с сохранением в БД
- Уроки: выбор темы → генерация AI с учётом слабых мест → 5 ответов → проверка → оценка
- Ошибки: структурированное извлечение (category|original|corrected|explanation), агрегация
- Прогресс: пройденные уроки, выученные слова, серия дней, топ слабых мест
- Напоминания: ежедневные, фильтрация по `onboarding_complete` + `reminder_enabled` + час
- Меню: план, ошибки, настройки (цель, минуты, напоминания, перетестирование)

**В работе (MVP 1):**
- Vocabulary review UX (SM-2 уже реализован, нужен UI-polish)
- Conversation flow (базовый чат есть, нужен управляемый режим)
- Alembic миграции (модели есть, `alembic init` не выполнен)
- Тесты
